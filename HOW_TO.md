# คู่มือการใช้งาน Markdown Viewer (HOW_TO.md)

ยินดีต้อนรับสู่ **Markdown Viewer** — แอปพลิเคชันอ่านไฟล์ Markdown แบบออฟไลน์ที่มีดีไซน์พรีเมียม สวยงาม ปลอดภัย และให้พื้นที่การทำงานสูงสุดบนหน้าจอของคุณ โดยข้อมูลทั้งหมดจะถูกประมวลผลอยู่บนเครื่องคอมพิวเตอร์ของคุณเท่านั้น ไม่มีการส่งข้อมูลขึ้น Cloud แต่อย่างใด

---

## 🌟 ฟีเจอร์เด่น (Key Features)

1. **ปลอดภัยสูง & ออฟไลน์ 100% (Offline-First Sandbox):**
   - ประมวลผลและอ่านไฟล์ด้วยระบบความปลอดภัยขั้นสูง (Strict Sanitization)
   - บล็อกสคริปต์ (Scripts), โค้ด HTML ที่ไม่ปลอดภัย และไฟล์รูปภาพภายนอกจากลิงก์ URL อัตโนมัติ เพื่อป้องกันการติดตามและการรันโค้ดอันตราย
2. **ขยายพื้นที่การอ่านเอกสารสูงสุด (Maximum Workspace):**
   - แถบเมนูด้านบน (Top Navigation Header) แบบมินิมอล ช่วยประหยัดพื้นที่แนวตั้งและแนวนอน
   - กล่องเอกสารกว้างสูงสุดถึง `1320px` พร้อมระบบจัดคำและตัดบรรทัดอัตโนมัติ (Word Wrap) ไม่เกิดปัญหาตัวอักษรหลุดขอบจอ
3. **ระบบสถิติการอ่านอัจฉริยะ (Document Stats):**
   - วิเคราะห์ขนาดไฟล์, จำนวนคำ (Word Count), จำนวนอักขระ (Character Count) และคำนวณเวลาที่ใช้ในการอ่าน (Estimated Read Time) ทันทีที่โหลดไฟล์
4. **บันทึกประวัติการเข้าใช้งานล่าสุด (Recent Documents Dashboard):**
   - บันทึกไฟล์ที่เคยเปิดล่าสุดสูงสุด 10 ไฟล์ไว้บนเครื่องคอมพิวเตอร์ (เก็บไว้ใน LocalStorage และ IndexedDB) ทำให้สามารถเปิดอ่านซ้ำได้ทันทีแม้ไม่ได้เชื่อมต่ออินเทอร์เน็ต
   - สามารถเลือก **ลบประวัติเฉพาะบางไฟล์** (ด้วยไอคอนถังขยะ) หรือกดล้างประวัติทั้งหมดได้
5. **ความสามารถในการเรนเดอร์เอกสารขั้นสูง (Rich Rendering):**
   - รองรับ **GitHub Flavored Markdown (GFM)** (เช่น ตาราง, รายการ Checklist, ลิงก์)
   - เรนเดอร์แผนภูมิโครงสร้างหรือไดอะแกรมด้วย **Mermaid.js** (เช่น Flowcharts, Sequence Diagrams, Mindmaps)
   - โค้ดบล็อกไฮไลต์สีสันสวยงาม (Syntax Highlighting) รองรับหลายภาษาคอมพิวเตอร์ และปุ่มกด **Copy Code** ที่มีแอนิเมชันตอบสนองเรียลไทม์
6. **ฟังก์ชันสะดวกเสริม:**
   - **Copy Raw:** คัดลอกเนื้อหา Markdown ดิบทั้งหมดได้ในคลิกเดียว
   - **Print PDF:** สั่งพิมพ์หน้าเอกสารออกทางเครื่องพิมพ์ หรือบันทึกเป็นไฟล์ PDF ได้ทันทีด้วยเค้าโครงเอกสารที่ถูกปรับแต่งมาให้เหมาะสมกับการพิมพ์โดยเฉพาะ (Print CSS)
   - **Theme Toggle:** สลับใช้งาน ธีมสว่าง (Light) และ ธีมมืด (Dark) สุดพรีเมียมได้อย่างรวดเร็ว

## 💻 การติดตั้งและเปิดใช้งานระบบ (Installation & Setup)

โปรเจกต์นี้สร้างขึ้นด้วย **React + TypeScript** และรันระบบอย่างรวดเร็วด้วย **Vite**

### 1. ความต้องการของระบบ (Prerequisites)
- ติดตั้ง **Node.js** (เวอร์ชัน 18 ขึ้นไปแนะนำ)
- ติดตั้งตัวจัดการแพ็คเกจอย่าง npm (ซึ่งติดตั้งมาพร้อม Node.js อยู่แล้ว)

### 2. ขั้นตอนการติดตั้ง (Installation)
เปิด Command Prompt หรือ Terminal ขึ้นมาในโฟลเดอร์โปรเจกต์นี้ จากนั้นพิมพ์คำสั่ง:

```bash
# 1. ติดตั้ง Dependencies ทั้งหมดของโปรเจกต์
npm install
```

### 3. การรันระบบบนเครื่องคอมพิวเตอร์ (Development Server)
หลังจากติดตั้งแพ็คเกจเสร็จสิ้น ให้ใช้คำสั่งนี้เพื่อรันเวอร์ชันสำหรับการทดสอบ/พัฒนา:

```bash
# 2. เริ่มต้นรันเซิร์ฟเวอร์จำลองโลคอล
npm run dev
```
หลังจากรันคำสั่ง ระบบจะขึ้นลิงก์สำหรับเข้าชม เช่น `http://localhost:5173` ให้คุณเปิดเบราว์เซอร์ไปที่ลิงก์ดังกล่าวเพื่อเข้าใช้งาน

### 4. การคอมไพล์โปรเจกต์เพื่อพร้อมใช้งานจริง (Build Production)
หากต้องการส่งออกเป็นโค้ดเว็บเพจหน้าเดียว (Single Page HTML) ที่พร้อมใช้งานอย่างมีประสิทธิภาพสูงสุด:

```bash
# 3. คอมไพล์โปรเจกต์ให้เป็น Static Assets
npm run build
```
โค้ดที่ผ่านการย่อขนาด (Minified) และบีบอัดจะไปอยู่ในโฟลเดอร์ `dist` ซึ่งคุณสามารถนำไปเปิดดูหรือใช้งานได้ทันที

## 📖 วิธีการใช้งานเบื้องต้น (Quick Start Guide)

1. **เปิดหน้าแรกของเว็บแอปพลิเคชัน:**
   - คุณจะพบกับหน้าต่างแดชบอร์ดสะอาดตา มีโลโก้ "Markdown Viewer" และปุ่มสลับธีมขวาบน
2. **การโหลดไฟล์:**
   - ลากไฟล์นามสกุล `.md`, `.markdown` หรือ `.txt` จากคอมพิวเตอร์ของคุณมาวางบนพื้นที่สีขาวสลัว (Drag & Drop zone)
   - หรือกดปุ่ม **Browse Files** เพื่อค้นหาและเลือกไฟล์ในระบบปฏิบัติการของคุณ
3. **การเข้าดูประวัติย้อนหลัง:**
   - หากคุณเคยเปิดไฟล์มาก่อน ประวัติไฟล์จะปรากฏในแท็บ **Recent documents** ด้านขวามือ คุณสามารถกดเปิดอ่านได้ทันที หรือกดไอคอนถังขยะเพื่อลบออกจากการจดจำ
4. **เมื่อเปิดเอกสารสำเร็จ:**
   - เนื้อหาจะแสดงตรงหน้าต่างหลักขยายกว้างเต็มตา 
   - แถบ Header ด้านบนจะโชว์สถิติของเอกสาร
   - คุณสามารถแก้ไข/สลับธีม หรือสั่งการกด **Copy Raw** / **Print** ได้ที่ปุ่มคำสั่งมุมบนขวา
   - เมื่อต้องการเปลี่ยนไฟล์ หรือกลับไปยังหน้าจอหลัก ให้กดปุ่ม **Back** (ลูกศรย้อนกลับ) ที่มุมซ้ายบน เพื่อกลับสู่หน้าแดชบอร์ดเริ่มต้น
